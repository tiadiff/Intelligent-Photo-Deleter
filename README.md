## ✨ Exclusive Features

### 🖼️ "Cozy" Visual Experience
- **Ambilight Effect (Ambient Glow)**: Photos project a dynamic luminous glow based on their colors, making viewing less tiring and more immersive.
- **Modern Dark Interface**: A clean design with rounded corners and contrasting colors for maximum comfort.
- **Smart Zoom**: Hover your mouse over an image to instantly activate a high-definition magnifying glass.

### ⚡ Top-Level Productivity
- **Ultra-Fast Navigation**: Scroll through photos with the arrow keys without any loading lag.
- **Smart Trash**: Move photos to a local subfolder without deleting them permanently, for total safety.
- **Instant Undo (Ctrl+Z)**: Made a mistake? Restore the last deleted photo in a millisecond.
- **Session Resume**: The program remembers where you left off and offers to resume from the exact photo of the last session.

### 📽️ Extended Format Support
- **Photos**: JPG, PNG, BMP, WebP, and native support for **HEIC/HEIF** (iPhone).
- **Videos**: Integrated support for MP4, MOV, and AVI with automatic loop preview via Edge WebView2.

---

## 🛠️ Tech Stack

- **Language**: VB.NET
- **Framework**: .NET 8.0 (Windows Forms)
- **Video Rendering**: Microsoft.Web.WebView2
- **HEIC Images**: Windows Imaging Component (WIC)
- **Storage**: JSON for session state saving

---

## 🚀 Getting Started

1. **Requirements**: Make sure you have the .NET 8 Runtime and the [WebView2 Runtime](https://developer.microsoft.com/en-us/microsoft-edge/webview2/) installed.
2. **Download**: Download the latest release or clone the repository.
3. **Execution**: Launch `photo-deleter.exe`, select your folder full of photos, and enjoy the cleanup!
